# VARS Project (E-Waste Powered Refrigeration System)

## Overview
MATLAB-based thermodynamic model of a vapor absorption refrigeration system powered by E-waste incineration.

## Key Results
- COP (actual): 0.75  
- COP (theoretical): 1.19  
- Cooling capacity: 1 TR

## Objective
Energy system modelling and waste-to-energy integration study.
